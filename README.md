# Order Management System

## Overview
A backend system for managing customers, products, and orders built with Node.js, Express.js, MySQL, and Sequelize ORM.

## Features
- Customer Management (Create, List)
- Product Management (Create, List)
- Order Management (Create, View, Cancel)
- Stock Management with Transactions
- Data Consistency with Database Transactions

## Tech Stack
- Node.js with Express.js
- MySQL Database
- Sequelize ORM

## Installation

### Prerequisites
- Node.js (v14 or higher)
- MySQL (v5.7 or higher)
- npm or yarn
